Intentional Typography

A series of typographic system and variable font demonstration pages

These pages are intended to showcase both typographic techniques applied to the web and the emerging Variable Font format. The typefaces featured are courtesy of their respective designers as noted. It would be remiss of me to not call out CJ Dunn and David Jonathan Ross in particular for their enthusiasm and generosity in beginning these demonstrations.

If you're interested in contributing or having your own variable fonts featured, please get in touch or contribute a pull request to the repository on Github. The fonts are not included; they are being served from my website but can be included from other sources as needed.

With regard to the German language pages: if you find spelling issues, please let me know so I can correct them! I've included the text in order to make an example case for different typographic treatments based on language.
